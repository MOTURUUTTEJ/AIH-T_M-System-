# AIH-T_M-System

## Project Overview
The AIH-T_M-System is designed to streamline hackathon management, featuring components such as AWS Explorer, Hackathon-Tracker, and a comprehensive landing page for hackathon management.

## Features
- **AWS Explorer:** Navigate and manage AWS services effortlessly.
- **Hackathon Tracker:** Monitor hackathon progress and participation.
- **Management Landing Page:** An intuitive interface for managing events and participants.

## Tech Stack
- **JavaScript:** 73.5%
- **CSS:** 17.2%
- **HTML:** 8.1%
- **Shell:** 1.2%

## Installation
1. Clone the repository: `git clone https://github.com/MOTURUUTTEJ/AIH-T_M-System`
2. Navigate to the project directory: `cd AIH-T_M-System`
3. Install dependencies: `npm install`

## Usage
- Start the application: `npm start`
- Open your browser and navigate to `http://localhost:3000` to access the system.
